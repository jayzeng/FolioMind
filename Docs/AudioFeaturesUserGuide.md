# FolioMind Audio Features - User Guide

## Quick Start

### Recording Your First Audio Note

1. **Start Recording**
   - Tap the "Record" button in the Audio Notes section
   - Grant microphone permission if prompted

2. **While Recording**
   - Watch the live audio level meter to ensure good input
   - See the duration counter update in real-time
   - **Pause**: Tap the pause button if you need a break
   - **Resume**: Tap play to continue recording
   - **Stop**: Tap the square stop button when finished

3. **After Recording**
   - Your note is automatically saved
   - Transcription begins in the background
   - Summary is generated once transcription completes

## Playback Controls

### Basic Playback
- **Play/Pause**: Tap the play button on any note
- **Progress**: Watch the circular progress indicator
- **Stop**: Tap another note or the stop button

### Advanced Playback (Detail View)
- **Seek**: Drag the slider to any position
- **Skip Forward**: Jump ahead 15 seconds
- **Skip Backward**: Go back 15 seconds
- **Speed Control**: Tap the speed menu to select:
  - 0.5× (half speed for detailed listening)
  - 0.75× (slower for note-taking)
  - 1.0× (normal speed)
  - 1.25× (slightly faster)
  - 1.5× (faster review)
  - 2.0× (double speed for quick review)

### Lock Screen Controls
- Play/pause, skip, and seek without opening the app
- See note title and duration
- Full media controls on lock screen and Control Center

## Organization Features

### Favorites
**Mark important recordings for quick access**
- **Star a note**: Tap the star in the menu or swipe left
- **View favorites**: Use the "Favorites" filter
- **Unstar**: Tap the star again or swipe left

### Tags
**Categorize notes for easy filtering**
- **Add tags**: Open note details → Tags section → "Add tag"
- **Remove tags**: Tap the × on any tag
- **Common tags**: "Meeting", "Idea", "Reminder", "Important"
- **View tagged**: Use the "Tagged" filter

### Folders (Optional)
- Set a folder name in note metadata
- Organize related notes together
- Filter by folder in search

## Search & Filter

### Searching
1. Tap the search bar
2. Type any text:
   - Searches note titles
   - Searches transcripts
   - Searches tags
3. Results update in real-time

### Filtering
Use the filter chips at the top:
- **All**: Show everything
- **Favorites**: Only starred notes
- **Recent**: Notes from the last 7 days
- **Tagged**: Notes with at least one tag

### Combining Search + Filter
- Select a filter first
- Then use search to narrow results
- Example: Filter "Favorites" → Search "meeting"

## Advanced Features

### Bookmarks
**Mark important moments in recordings**

1. While playing a note:
   - Tap menu (•••) → "Add Bookmark"
   - Add an optional note (e.g., "Important decision at 3:45")
   - Tap "Add"

2. Jump to bookmarks:
   - See all bookmarks in the detail view
   - Tap any bookmark to jump to that timestamp
   - Delete bookmarks with the trash icon

### Transcript Editing
**Fix transcription errors**

1. Open note details
2. Find the "Transcript" section
3. Tap "Edit"
4. Make your changes
5. Tap "Done" to save

### Renaming Notes
**Give descriptive titles**

1. Open note details
2. Tap menu (•••) → "Rename"
3. Enter new title in navigation bar
4. Tap outside or press Enter to save

## Batch Operations

### Select Multiple Notes
1. Tap "Select" button

3. Choose action:
   - **Toggle Favorites**: Star/unstar all selected
   - **Delete Selected**: Remove multiple notes

4. Tap "Cancel" to exit selection mode

## Tips & Tricks

### Recording Tips
1. **Pause for Interruptions**: Don't restart - just pause!
2. **Watch the Meter**: Green is good, yellow is okay, red might clip
3. **Quiet Environment**: Better transcription quality

### Playback Tips
1. **Speed Up Reviews**: Use 1.5× or 2× for quick listening
2. **Slow Down**: Use 0.75× for detailed note-taking
3. **Skip Around**: Use 15s skip buttons for quick navigation

### Organization Tips
1. **Tag Immediately**: Add tags right after recording
2. **Use Consistent Tags**: Stick to a few categories
3. **Favorite Important**: Star notes you'll need again
4. **Search Transcripts**: Find specific mentions without listening

### Workflow Examples

**Meeting Notes**
1. Record during meeting (pause when off-topic)
2. Add tags: "Meeting", "Project X"
3. Star if important decisions were made
4. Add bookmarks for action items
5. Edit transcript to fix names/terms
6. Share recording with team

**Voice Memos**
1. Quick record idea
2. Tag: "Idea", "Todo"
3. Star if urgent
4. Review at 2× speed later
5. Add to task manager if actionable

**Interview Recording**
1. Record entire interview
2. Add bookmarks for key quotes
3. Tag with interviewee name
4. Use 0.75× speed for transcription review
5. Edit transcript for accuracy
6. Share recording and transcript

## Troubleshooting

### Transcription Failed
- **Retry**: Use "Retry transcription" in menu
- **Check permissions**: Settings → FolioMind → Speech Recognition
- **File quality**: Better audio = better transcription

### Can't Hear Playback
- Check device volume
- Try different playback speed
- Verify file exists (doesn't show "missing")

### Recording Not Starting
- **Permissions**: Settings → FolioMind → Microphone
- **Try restarting**: Close and reopen app
- **Check storage**: Ensure enough space
